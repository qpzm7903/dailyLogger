### MAINT-005: 升级 jsdom

**背景**: jsdom 有新的 Major 版本 29.0.1，需要评估是否升级。

**风险**: 低风险，jsdom 是测试依赖，不影响生产代码。

### MAINT-006: 升级 vitest

**背景**: vitest 有新的 Major 版本 4.1.0，需要评估 Breaking Changes。

**风险**: 中等风险，需要验证所有测试在升级后仍然通过。

### MAINT-007: Tailwind CSS 4.x 迁移评估

**背景**: Tailwind CSS 4.x 是完全重写的版本，有重大变化。

**风险**: 高风险，需要评估迁移成本和收益。

**评估结论**: ⏸️ 暂缓迁移

**原因**:
1. Tailwind CSS 3.x 仍在维护中，无安全风险
2. 4.x 迁移涉及配置文件格式变更（JS → CSS）
3. 项目中有大量 Tailwind 类使用，迁移测试成本高
4. 当前版本工作稳定，无紧迫迁移需求

**后续计划**: 待 Tailwind CSS 4.x 生态更成熟后再评估迁移
