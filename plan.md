## v2.1.0（核心价值深挖）✅ 完成

**目标**: 深耕 AI 工作记忆核心能力，从"截图记录"进化为"理解工作"

| Phase | 功能 | 价值 | 状态 |
|-------|------|------|------|
| 1 | 自定义 API Headers — 支持 OpenRouter/Azure/Claude 等 API 服务 | 解决 Issue #68，扩展 API 兼容性 | ✅ 完成 |
| 2 | 智能窗口上下文 — 结合窗口标题 + 活跃应用判断工作类型 | 从"看图说话"升级为"理解工作" | 规划中 |
| 3 | 工作模式检测 — 自动识别编码/会议/文档/沟通等模式 | 日报从"做了什么"进化为"怎么分配时间的" | 规划中 |
| 4 | 跨天关联分析 — 相似工作内容自动关联，形成项目维度 | 周报/月报质量飞跃 | 规划中 |
| 5 | 洞察仪表板 — 可视化工作模式趋势，提供效率建议 | 从"记录工具"变成"效率教练" | 规划中 |
