## 风险与缓解

| 风险 | 影响 Epic | 缓解措施 |
|-----|----------|---------|
| Vision API 成本高 | Epic 2, 3 | 支持本地模型，优化调用频率 |
| 跨平台兼容性 | Epic 2 | 充分测试各平台截图功能 |
| 大数据量性能 | Epic 4 | 分页加载，索引优化 |
| 集成复杂度 | Epic 6 | 优先支持成熟 API |
