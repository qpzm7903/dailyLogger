## Lessons Learned

### Technical

1. **模块删除检查清单**
   - 问题：删除功能时容易遗漏关联代码
   - 解决：使用 Grep 搜索关键词（如 "github"）确保彻底清理
   - 建议：建立删除检查清单：后端模块 → 前端组件 → 类型定义 → 设置字段 → 测试文件 → 国际化 → 文档

2. **类型定义同步**
   - 问题：前端 TypeScript 类型需要与后端 Rust 结构体保持同步
   - 解决：逐字段对比，确保两端一致删除
   - 建议：考虑使用代码生成工具（如 ts-rs）保持类型同步

3. **模板占位符处理**
   - 问题：日报模板中的 `{github_activity}` 占位符需要处理
   - 解决：替换为空字符串，保留注释说明历史原因
   - 建议：模板清理应纳入删除流程
