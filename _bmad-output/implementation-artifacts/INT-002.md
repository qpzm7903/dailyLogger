# Story 6.2: Logseq 导出支持

Status: ready-for-dev

## Story

As a DailyLogger 用户,
I want 将工作日报自动导出到 Logseq 图谱,
so that 我可以在 Logseq 中利用图谱功能管理和回顾工作记录。

## 背景

Logseq 导出功能已有基础实现：
- 后端 `write_report_to_logseq` 函数已存在于 `synthesis/mod.rs`
- 数据库 `logseq_graphs` 字段已支持多图谱配置
- 前端 `OutputSettings.vue` 已有 Logseq 图谱管理 UI

本 Story 需要验证现有实现的完整性，确保功能正确工作，并补充可能缺失的测试和文档。

## Acceptance Criteria

1. 日报生成后，Markdown 内容正确写入 Logseq 图谱的 `pages` 目录
2. 支持多个 Logseq 图谱配置，用户可选择默认图谱
3. 前端 UI 支持添加/删除/设置默认图谱
4. 周报、月报、自定义报告也正确导出到 Logseq
5. 现有单元测试覆盖核心功能
6. 用户文档更新，说明 Logseq 配置方法
