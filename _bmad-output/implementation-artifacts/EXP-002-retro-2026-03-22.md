## Lessons Learned

### Technical

1. **图像熵算法选择**
   - 熵值范围 0~8（log2(256)），归一化后映射到 0.0~1.0
   - 纯色图像熵接近 0，丰富内容熵高
   - 32x32 缩放平衡了计算效率和质量判断准确性

2. **数据库迁移模式**
   - 新增字段：`quality_filter_enabled INTEGER DEFAULT 1`
   - 新增字段：`quality_filter_threshold REAL DEFAULT 0.3`
   - 使用 DEFAULT 确保旧数据库自动兼容

3. **原子计数器使用**
   - 使用 `std::sync::atomic::AtomicU32` 实现线程安全计数
   - 统计为当日内存计数，应用重启自动重置
