## Lessons Learned

### Technical

1. **功能发现模式**
   - 开发前检查现有实现可大幅减少工作量
   - `ScreenshotModal.vue` 已有功能可作为参考模板
   - Story 文档中的"已实现/本 Story 补充"分区非常有价值

2. **组件通信模式**
   - `ScreenshotGallery` → `invoke` → 后端 → 返回结果 → 更新本地状态
   - 简单场景无需引入复杂状态管理
   - 直接更新 `screenshots` 数组即可触发 UI 刷新

3. **事件处理细节**
   - 卡片上按钮需 `@click.stop` 阻止冒泡
   - 避免触发卡片点击打开详情弹窗

- Story 文档"背景"部分清晰区分已实现/待实现，避免重复工作
- Dev Notes 中提供代码参考模板，加速开发
