## Lessons Learned

### Technical

1. **组件通信模式**
   - 问题：TimelineWidget 的 `refresh()` 方法暴露后，父组件需要显式调用
   - 解决：在 Dashboard.vue 中添加 watch 监听数据变化并调用 refresh
   - 建议：设计组件 API 时考虑响应式数据流，优先使用 props/reactive 而非手动方法调用

2. **Git 提交文件追踪**
   - 问题：`Cargo.lock` 文件变更未记录在 Story File List 中
   - 解决：已记录为非功能影响
   - 建议：提交时检查 `git diff --name-only` 确保所有变更文件都被记录
