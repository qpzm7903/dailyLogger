# Story 3.3: 自定义日报模板

Status: done

## Story

作为一个 DailyLogger 用户，
我希望能够自定义日报生成的 Prompt 模板，选择预设模板或导入自定义模板，
以便让生成的日报更符合我的工作场景和团队规范，同时可以备份和分享我的模板配置。

## Acceptance Criteria

### AC1 - 支持编辑 summary_prompt
- Given 用户打开设置界面
- When 查看"日报生成"部分
- Then 显示"日报 Prompt"文本框，可编辑自定义 Prompt
- Given 用户输入自定义 Prompt 并保存
- When 生成日报时
- Then AI 使用自定义 Prompt 生成日报
- Given 用户清空 Prompt 后保存
- When 生成日报时
- Then AI 使用默认 Prompt 进行生成

### AC2 - 提供多种模板选择
- Given 用户打开设置界面
- When 点击"模板库"按钮
- Then 显示预设模板列表（至少 3 种）
- Given 用户选择一个预设模板
- When 点击"应用"
- Then summary_prompt 字段填充为该模板内容
- And 显示"已应用模板 X"提示

### AC3 - 支持导入/导出模板
- Given 用户已配置自定义 Prompt
- When 点击"导出模板"按钮
- Then 生成 JSON 文件下载，包含模板名称和内容
- Given 用户点击"导入模板"按钮
- When 选择有效的 JSON 模板文件
- Then summary_prompt 字段填充为导入的内容
- And 显示"导入成功"提示
- Given 用户导入无效文件
- When 解析失败
- Then 显示具体错误信息

### Prompt 占位符说明

模板中必须包含 `{records}` 占位符，系统会替换为当日工作记录：
```
今日记录：
{records}
```
