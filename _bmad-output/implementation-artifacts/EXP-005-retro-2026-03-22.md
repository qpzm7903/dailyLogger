## Lessons Learned

### Technical

1. **SQL 聚合函数的 NULL 处理**
   - SQLite 的 `SUM()` 在空结果集上返回 NULL，非 0
   - 使用 `COALESCE()` 或 `IFNULL()` 是标准解决方案
   - 建议在所有聚合查询中考虑此边界情况

2. **前端状态持久化模式**
   - `localStorage` 适合存储简单的用户偏好（如折叠状态）
   - 无需引入额外的状态管理库
   - Key 命名建议包含组件前缀避免冲突

3. **Widget 与主界面联动模式**
   - 通过 `watch` 监听 `todayRecords` 变化触发刷新
   - 单一数据源，避免多个轮询实例
   - 响应式设计比手动轮询更优雅
